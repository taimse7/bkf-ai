# BKF AI

אפליקציית macOS ב־Tauri 2, Rust, React ו־TypeScript לסריקת ספריית קובצי BKC/BKF,
ניהול ספרייה מקומית ותור המרות מוגבל ליכולות שנבדקו.

מצב האימות והמענה לדוח הפיקוח מתועדים ב־`AUDIT_RESPONSE_HE.md`, ב־`PROGRESS.md`
וב־`KNOWN_LIMITATIONS.md`. לאחר כל העלאה ל־GitHub יש להפעיל את workflow
`Mac Build Proof`; רק ריצה ירוקה מוכיחה שה־backend נבנה ושנוצרו `.app` ו־`.dmg`.
