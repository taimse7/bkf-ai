# מגבלות ידועות

1. אין מפענח BKF עצמאי וכללי.
2. תצוגת BKF דורשת Sidecar תואם וגם Renderer של DjVu; בפרויקט קיימת תשתית
   לשחזור עמוד, אך Renderer מלא אינו כלול.
3. המרת BKC מדויקת קיימת רק לפרופיל המאומת `674817`.
4. וריאנט BKC לא מוכר משתמש ב־Ghostscript אם הוא מותקן וזמין.
5. Ghostscript אינו כלול בחבילה.
6. חילוץ טקסט מ־PDF הוא Best Effort ואינו OCR.
7. BKF searchable רק כאשר קיים Text Sidecar או ספק טקסט עתידי.
8. ה־API המקומי משתמש כרגע בפורט קבוע `47831`.
9. Token החיבור לתוסף מוזן ידנית בגרסה זו.
10. תצוגת PDF בתוסף אוצריא משתמשת ב־iframe של PDF שהשרת מספק; אפליקציית
    ה־Desktop משתמשת ב־PDF.js.
11. Range Requests נתמכים עבור PDF; בדיקות עומס על ספרים גדולים עדיין נדרשות.
12. חתימה ו־Notarization אינן מוגדרות.
13. לא בוצעה בסביבה זו קומפילציית Rust או Tauri Native; יש להסתמך על GitHub
    Actions לאחר העלאת הפרויקט.
