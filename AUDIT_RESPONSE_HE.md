# מענה לדוח הפיקוח — 21 ביולי 2026

## מה תוקן בחבילת המקור הזאת

1. רכיב `bkf-container-probe` מחובר כתלות ישירה של ה־backend הפעיל.
2. הפקודות `probe_book_structure` ו־`export_probe_report` רשומות ב־Tauri
   `generate_handler!` ולכן נגישות לממשק.
3. בממשק React קיים כפתור „בדיקת מבנה” לכל קובץ וכפתור „הורדת דוח JSON” לאחר בדיקה.
4. ה־workflow מריץ גם את בדיקות ה־backend וגם את בדיקות `container-probe` לפני בניית DMG.
5. התיעוד אינו מציג עוד commit מקומי ישן כאילו הוא ה־HEAD הנוכחי של GitHub.

## מה עדיין אינו מוכח

- טרם קיימת בחבילת המקור עצמה הוכחה שריצת GitHub Actions על ה־commit שיועלה הצליחה.
- אין במאגר קובצי golden גדולים, ולכן תוצאות ההמרה שלהם אינן ניתנות לשחזור מן המאגר בלבד.
- אין ממיר BKF, אין ממיר BKC כללי ואין Ghostscript מוטמע בתוך ה־DMG.
- כלי ה־Probe הוא כלי ניתוח מבני בלבד ואינו מפענח תוכן.

## שער השחרור

לאחר העלאת החבילה ל־GitHub יש להפעיל `Mac Build Proof`. רק ריצה ירוקה שבה עברו
`pnpm build`, בדיקות Rust, בניית האפליקציה, אימות ה־artifacts והעלאתם נחשבת הוכחת build
לגרסה שהועלתה.
